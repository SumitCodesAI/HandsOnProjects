# React & Redux Learning Portal

A comprehensive, interactive learning platform for mastering React and Redux with hands-on examples and detailed notes.

## 🚀 Features

- **Interactive Demos**: Three fully functional Redux applications
  - Counter with multiple operations
  - Todo List with filtering
  - User Management with CRUD operations

- **Learning Content**: Comprehensive notes covering
  - React fundamentals and hooks
  - Redux Toolkit basics and best practices
  - State management patterns

- **Modern UI**: Beautiful, responsive design with gradient accents
- **Real-time State Management**: See Redux in action with live examples

## 📦 Technologies Used

- **React 19** - UI library
- **Redux Toolkit** - State management
- **React Router** - Navigation
- **Vite** - Build tool
- **CSS3** - Styling

## 🛠️ Installation

1. Clone the repository or navigate to the project directory

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## 🎯 Learning Path

1. **Start with Home** - Get an overview of the portal
2. **Read React Notes** - Learn React fundamentals
3. **Read Redux Notes** - Understand state management
4. **Try Demos** - Practice with interactive examples
   - Counter: Basic Redux operations
   - Todo List: Array state management
   - User Management: Full CRUD operations

## 🎨 Features by Demo

### Counter Demo
- Increment/Decrement
- Reset functionality
- Custom amount operations
- Visual state display

### Todo Demo
- Add/Delete todos
- Toggle completion
- Filter by status (All/Active/Completed)
- Clear completed items

### User Management Demo
- Add new users
- Edit user details
- Delete users
- Role management
- Responsive card layout

## 📖 What You'll Learn

- Component-based architecture
- React Hooks (useState, useEffect, useSelector, useDispatch)
- Redux Toolkit slices and store
- Immutable state updates
- Action creators and reducers
- Routing with React Router
- CSS layouts and responsive design
